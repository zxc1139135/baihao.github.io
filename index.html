Research Profile

My work aims to improve the security, efficiency, and privacy of machine learning systems. Recent studies include personalized federated learning under non-IID data, efficient privacy-preserving CNN inference, and differentially private federated learning with generative data augmentation.

  • Current focusMachine learning privacy protection and trustworthy AI systems
  • MethodsKnowledge distillation, secret sharing, secure aggregation, GAN-based data generation
  • Application scenarioCollaborative learning, privacy-preserving inference, and secure data analytics

Selected Work

Research

Federated Learning

ACE-pFL: Personalized Federated Learning with Knowledge Distillation

An accurate and efficient personalized federated learning framework that uses EMD-based client clustering, adaptive distillation temperature, and triple knowledge distillation to handle non-IID data.

IEEE Transactions on NetworkingCCF-A

Privacy-preserving ML

Efficient Privacy-preserving CNN Inference

A secret-sharing-based CNN inference scheme that separates linear and non-linear computation, using matrix split computation and polynomial approximation for efficient privacy protection.

Journal of SoftwareCCF-T1

Differential Privacy

GetFed: Differentially Private Federated Learning with GAN

A differentially private FL framework that combines GAN-based data generation, adaptive noise reduction, and secure aggregation for improved accuracy on non-IID data.

IEEE TDSCCCF-A
